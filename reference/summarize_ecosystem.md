# Calculate ecosystem rank based upon Atlantis/Ecosim output

Calculate ecosystem rank based upon Atlantis/Ecosim output

## Usage

``` r
summarize_ecosystem(ecosystem_data)
```

## Arguments

- ecosystem_data:

  R data objected created from a csv file with ecosystem scores by
  species based on Atlantis/Ecosim (data-raw/ecosystem_data.csv)
